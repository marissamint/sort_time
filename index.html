Guidance

(Highlights the current mode you have selected)

Nozzle

Establishes the diameter of the stream and size of the droplet.

Note: Larger cells require a larger nozzle; the 85um nozzle is appropriate for most common cell types.

Number of sorted cells per sample

The number of cells you would like to be sorted for each sample.

Note: This is not the total number of cells, just the number you want sorted into the collection tube.

Expected Event Rate (evt/s)

The expected threshold rate for sorting.

Note: Typically kept below 5,000 evt/s. Use the Event Rate Calculator for an estimate.

Sort Efficiency (%)

Percentage of selected cells that were successfully sorted.

Note: A lower event rate, smaller nozzle, and higher population frequency can improve efficiency.

Sort Rate (evt/s)

The number of cells sorted per second.

Estimated minimum sort time per sample

The minimum time required to sort one sample.

Note: A lower event rate or population frequency will increase sorting time.

Sample Concentration (cells/ml)

The concentration of cells in your sample.

Flow Rate

A numerical value that determines the rate a sample is processed.

Note: A higher flow rate increases event rate but may stress cells.

Estimated True Flow Rate (uL/s)

An estimate of the instrument's actual flow rate.

Estimated Event Rate (evt/s)

The approximate event rate based on sample concentration and flow rate.

Note: Dead cells or debris not counted in the sample can lead to a higher actual event rate.
